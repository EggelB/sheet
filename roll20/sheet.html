Casting Quick Reference

  • Sorcery Test: Sorcery skill + Spell Pool dice vs Target Number
  • Default TN: Spell Force (or target's attribute — varies by spell)
  • Drain Resist: Willpower + remaining Spell Pool dice vs Drain TN
  • Drain TN: Derived from Drain Code (e.g. M+1 = Force/2 +1, S = Force, etc.)
  • Sustaining Penalty: +2 TN to ALL actions per spell sustained
  • Foci: Sustaining focus removes the +2 TN penalty for that spell
